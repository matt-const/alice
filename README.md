# ALICE Engine

![Build](https://github.com/matt-const/alice/actions/workflows/ci.yml/badge.svg)

Aims to be a simple, minimal engine for building terminal / graphical applications in C.
Works out of the box, with minimal setup.

## Build Instructions.
### Windows
  Requires MSVC to be installed and `cl.exe` to be in your $PATH.  
  Build with: ``./build.bat``

### MacOS
  Requires `clang` from the xcode command line tools.  
  Build with: ``./build.sh``

### Linux
  Requires `clang` to be installed.  
  Build with: ``./build.sh``
